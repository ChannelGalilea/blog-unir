TEMA 2. INTRODUCCIÓN A HOJAS DE ESTILO CSS

En la creación de páginas web, el lenguaje HTML únicamente se encarga de estructurar el contenido. Cuando a ese contenido es necesario cambiarle la apariencia visual se utilizan hojas de estilos en cascada o CSS (Cascade Style Sheets).

El objetivo principal de las hojas de estilo es lograr una separación total entre el contenido y la estructura de su estilo de presentación. Sin embargo, el lenguaje HTML permite la inclusión de estilos en la propia creación del documento HTML (aunque no se trata de una buena práctica según el W3C).

El funcionamiento de las hojas de estilo permite definir la apariencia de una web en diferentes niveles:

  • A nivel local utiliza una sintaxis para realizar una modificación que solo afecta a un punto del documento.
  • A nivel de página utilizando la cabecera.
  • A nivel de archivos externos.

Desde el punto de vista del diseño de la web se recomienda utilizar hojas de estilos externas, pues son más fáciles de mantener y un cambio en sus reglas afectará a todas las páginas implicadas.
